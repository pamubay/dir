**`cache_dir`**

- Fork:
  [justjavac/deno_dirs/cache_dir](https://github.com/justjavac/deno_dirs/blob/e8c001bbef558f08fd486d444af391729b0b8068/cache_dir/mod.ts).

**`deno_dir`**

- Set Deno cache directory
